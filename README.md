# 🥖 Bakery Landing Page

A modern, responsive, and visually appealing Bakery Landing Page built using HTML, CSS, and JavaScript. This project showcases a professional bakery website with elegant design, smooth navigation, delicious product sections, and a fully responsive user experience across all devices.

## 🚀 How to Run Locally

1. Clone the repository:

```bash
git clone https://github.com/roghayem/Bakery-landing-page.git
```

2. Open the project folder:

```bash
cd Bakery-landing-page
```

3. Open `index.html` in your browser.
